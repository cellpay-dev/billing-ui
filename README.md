# CellPay Portal - Business Management Platform

## 🎯 Overview

**CellPay Portal** is a comprehensive business management platform designed for B2B and D2C payment processing operations. Built with modern web technologies and optimized for Cloudflare Pages deployment.

---

## ✨ Features

### **📊 Real-Time Dashboard**
- 4 key performance indicators (KPIs) with trend arrows
- 4 interactive charts (Revenue Trend, Transaction Volume, Top Carriers, Payment Methods)
- B2B vs D2C performance comparison
- Sales drop alerts with customer details
- Auto-refresh every 5 minutes

### **👥 Employee Management**
- Complete CRUD operations (Create, Read, Update, Delete)
- Role-Based Access Control (RBAC): Admin, Manager, Employee
- Manual password setup with admin control
- Password reset functionality
- Employee status management (Active, Suspended, Inactive)
- Department organization

### **🏢 CRM System**
- **B2B Customer Management**

  - ISO/Sub-ISO mapping

## ⚙️ Configuration Management

The application uses a centralized configuration system for easy environment management.

### **Configuration Files**
- **`public/static/config.js`** - Main configuration (API URLs, timeouts, feature flags)
- **`public/static/api.js`** - REST API layer (reads from config.js)
- **`public/static/app.js`** - Main application (uses config values)

### **Key Configuration Parameters**
```javascript
window.AppConfig = {
  api: {
    backendUrl: 'https://uat.cellpaybackend.us:8443/reports/api',
    timeout: 30000
  },
  auth: {
    tokenKey: 'token',
    userKey: 'user'
  },
  app: {
    debug: true,
    dashboardRefreshInterval: 1800000  // 30 minutes
  }
}
```

### **Environment-Specific Setup**
Edit `public/static/config.js` to set different backends for dev/staging/production:
```javascript
// Development
if (window.location.hostname === 'localhost') {
  window.AppConfig.api.backendUrl = 'http://localhost:8080/api';
}

// Production
if (window.location.hostname === 'cellpay.com') {
  window.AppConfig.api.backendUrl = 'https://api.cellpay.com';
}
```
